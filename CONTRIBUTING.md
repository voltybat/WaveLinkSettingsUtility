# Contributing

Issues and pull requests are welcome. Keep changes focused, add synthetic tests for behavior changes, and run `dotnet test -c Release` before submitting.

Never commit or share a real Wave Link `Settings.json`; create a minimal synthetic fixture instead. Do not add telemetry, network access, elevation requirements, or code that stops Wave Link/Elgato services.
